Nucleus Counter Site Plan

WDD 131-01

Overview

Purpose

To display and account for a total set of items from cumulative Nucleus Runs

Audience

The specific demographic within the game of Hypixel Skyblock that are looking to track their total yield from Nucleus Run Bundles

Dynamic elements

Javascript elements will react to user input either adding to or detracting from the total, display the changes, and store the totals within localhost
